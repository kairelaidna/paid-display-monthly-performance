# Paid Display Monthly Performance

GitHub Pages upload package for the Paid Display Monthly Performance tool.

## Files to Upload

Upload everything in this folder to the root of the GitHub repository:

- `.nojekyll`
- `index.html`
- `public/marketing-performance-slideshow.html`
- `public/assets/google-ac-logo.png`
- `public/assets/moloco-logo.png`
- `public/assets/the-trade-desk-logo.png`

## URL

After GitHub Pages updates, the tool can be opened from either:

```text
https://kairelaidna.github.io/paid-display-monthly-performance/
```

or:

```text
https://kairelaidna.github.io/paid-display-monthly-performance/public/marketing-performance-slideshow.html
```

## Security

The API key is not stored in these files. It is pasted into the browser when the report is generated.

The source Google Sheets stay Wise-protected and are read through the Wise Apps Script data bridge configured inside `public/marketing-performance-slideshow.html`.
